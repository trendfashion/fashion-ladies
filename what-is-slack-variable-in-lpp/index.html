What Is Slack Variable In Lpp

Written by Petter Sep 03, 2023 · 5 min read

Have you ever heard of slack variables in Linear Programming Problems (LPP)? If you're not familiar with them, don't worry, you're not alone. In this tutorial blog post, we will explain what slack variables are in LPP, why they are important, and how they can help you optimize your LPP.

If you're struggling with optimizing your LPP or finding the most efficient solution, you're not alone. Many people find LPP challenging, and it can be overwhelming to try to figure it out on your own. Fortunately, slack variables can help simplify the process and make it easier to find the optimal solution.

Slack variables are variables that are added to LPP to make the constraints less than or equal to constraints. These variables are used to convert inequality constraints to equality constraints, which makes it easier to solve the problem using the simplex method.

To summarize, slack variables are added to LPP to convert inequality constraints to equality constraints, making it easier to solve the problem using the simplex method. By doing so, you can find the optimal solution to your LPP.

Why Are Slack Variables Important?

Slack variables are important because they allow us to convert inequality constraints to equality constraints. This conversion simplifies the problem and makes it easier to solve using the simplex method. Without slack variables, we would have to use the two-phase method, which is more complicated and time-consuming.

Furthermore, slack variables allow us to determine the feasibility of the solution. If the slack variables are non-negative, it means the solution is feasible. If they are negative, it means the solution is infeasible.

How Do Slack Variables Work?

Let's say we have an LPP with the following constraints:

5x + 2y ≤ 10

3x + 6y ≤ 15

To convert these constraints to equality constraints, we would add slack variables s1 and s2, respectively. Our constraints would then look like this:

5x + 2y + s1 = 10

3x + 6y + s2 = 15

By doing this, we have converted the inequality constraints to equality constraints. We can now use the simplex method to find the optimal solution to the LPP.

How To Solve LPP Using Slack Variables?

To solve an LPP using slack variables, follow these steps:

  1. Convert the inequality constraints to equality constraints by adding slack variables.
  2. Set up the initial simplex tableau.
  3. Choose the entering variable with the most negative coefficient in the objective function.
  4. Choose the leaving variable by finding the smallest non-negative ratio of the right-hand side of the constraint to the corresponding coefficient of the entering variable.
  5. Perform row operations to make the entering variable equal to 1 and all other coefficients in the column equal to 0.
  6. Repeat steps 3-5 until all coefficients in the objective function are non-negative.
  7. Read the optimal solution from the final simplex tableau.
